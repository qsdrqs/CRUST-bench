TwoBitHeader
This structure holds the fixed-sized file header (16 bytes, of which 4 are blank). The version should always be 0. In theory, the endianness of the magic number can change (indicating that everything in the file should be swapped). As I&#39;ve never actually seen this occur in the wild I&#39;ve not bothered implementing it, though it&#39;d be simple enough to do so.
